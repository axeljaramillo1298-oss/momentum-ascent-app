FUENTES

API o mock

Si no hay API key configurada, el sistema usa eventos mock realistas para no bloquear desarrollo local ni la demo del flujo completo.

COSTO

Control de llamadas

La IA no se ejecuta en cada carga del frontend. Primero se reutiliza el pick guardado y solo se genera uno nuevo cuando hace falta o cuando admin lo fuerza.
