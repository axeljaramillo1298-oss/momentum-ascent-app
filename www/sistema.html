Metodologia del producto

Como funciona el sistema

API deportiva, almacenamiento persistente, generación IA y consumo frontend en un flujo simple orientado a MVP.

1. Eventos

Se sincronizan desde API real o mock realista cuando no hay credenciales configuradas.

2. Base de datos

Los eventos, stats y picks quedan guardados para reutilización y contexto histórico.

3. IA

La IA genera un pick solo cuando hace falta y devuelve market, confianza, riesgo y análisis.

4. Frontend

El dashboard muestra picks del día, análisis e historial sin recalcular todo en cada carga.

5. Control de costo

Si ya existe pick reciente para el evento, se reutiliza en lugar de volver a llamar a la IA.

6. Disclaimer

Todo el producto se presenta como contenido informativo, no como promesa de ganancia.
